\doxysubsubsection{\texorpdfstring{saveToFile()}{saveToFile()}}
{\footnotesize\ttfamily void save\+To\+File (\begin{DoxyParamCaption}\item[{char \texorpdfstring{$\ast$}{*}}]{filename }\end{DoxyParamCaption})}



Saves the packages to a file. 

This function saves the information about each package in the hash table to a specified file.


\begin{DoxyParams}{Parameters}
{\em filename} & The name of the file to which packages will be saved.\\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
The file will be opened in write mode ("{}w"{}). If the file does not exist, it will be created.
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
If the file cannot be opened for writing, an error message will be printed, and the function will return without saving.
\end{DoxyWarning}
The package information will be saved in CSV (Comma-\/\+Separated Values) format, with each line representing a package and fields separated by commas. The order of fields is as follows\+:
\begin{DoxyItemize}
\item Tracking Number
\item Current Location
\item Estimated Arrival Time
\item Destination Address
\end{DoxyItemize}

Example CSV format\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{ABC123,Storage,2023-\/01-\/01,123\ Main\ St}
\DoxyCodeLine{XYZ789,In\ Transit,2023-\/01-\/05,456\ Oak\ Ave}

\end{DoxyCode}


\begin{DoxyWarning}{Warning}
If a package has special characters, the CSV format may need to be adjusted to accommodate those characters.
\end{DoxyWarning}
Example Usage\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{saveToFile(\textcolor{stringliteral}{"{}package\_data.csv"{}});}

\end{DoxyCode}
